# Cybersecurity-Programming-and-Networking-Project


## About This Project

This project was developed as an end-semester assignment for CYSE-250 Cybersecurity Programming and Networking. The goal was to design and implement a Python-based client-server system using socket programming.

As an online student, I chose to build a secure student–educator portal for a history-based learning environment focused on the Sierra Leone Civil War. The system allows users to register, log in, securely access encrypted documents, and participate in quizzes designed to reinforce learning.

The platform also includes role-based access control, activity logging, and administrative tools for educators to manage quiz content and monitor student performance.

This project demonstrates practical applications of networking, authentication, encryption, and secure software design using Python.
